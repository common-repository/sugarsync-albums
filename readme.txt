== Description ==

Your pictures in the Sugarsync albums, can be simple, quick insert into your blog post, and supports multiple display effect.The real External interfaces, do not need to install any client.
SugarSync appears to have a powerful sync network disk, you can synchronized any files from mobile devices or computer.
SugarSync also include a strongly album function, which limited to its sharing function. If you own numbers of photos or albums, the sharing process could be an endless and messy journey.
Now [codante.org](http://codante.org "codante.org") provides a much visual and easy way to solve the problem.

将您Sugarsync相册中的图片，简单、快捷的插入到您的博客文章中，并支持多种展示效果。真正的外部接口，不需要安装任何客户端。
SugarSync是一个功能强大的同步网盘，您可以将任何个人电脑或者移动设备上的文件进行同步。
SugarSync还拥有非常强大的相册功能，但SugarSync的相册分享有着很大的局限性，如果您拥有很多相册或者相册中有大量图片，您想把图片分享到您的网站、博客、社区或者其他地方将会是一个繁琐而又漫长的过程...
现在 [codante.org](http://codante.org "codante.org") 为您提供了更加直观而且操作简便的方法！

== Installation ==

1. Upload the entire "sugarsync-albums" folder to the `/wp-content/plugins/` directory.将文件夹上传到"/wp-content/plugins/"目录中。
2. Activate the plugin through the "Plugins" menu in WordPress.在wordpress的后台"插件"页面将本插件启用。
